# DevTool
====
Houdini常用工具插件

- https://github.com/TimeVfx/DevTool
### 插件
#### 1.PARMmenu.xml
##### Open Folder
鼠标右键快速打开Cache文件窗口


### 快速安装

- 下载PARMmenu.xml文件
- 将PARMmenu.xml拷贝到C:\Users\用户\Documents\houdiniXX文件夹下
- 重启Houdini

### 使用
- 启动Houdini
- Geometry层级下创建一个File Cache节点
- 输出路径栏（Geometry File）上鼠标右击，选择Operation->Open Folder
